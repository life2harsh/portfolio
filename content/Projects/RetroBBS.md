## Award
**First Runner-Up, OSDHack '25** — recognised by FOSS United, .xyz, and CodeCrafters

## Stack
`JavaScript` · `Python` · `WebSockets`

## What it does

- Low-latency **voice chat** and **real-time messaging** via WebSockets
- Retro BBS aesthetic with a modern real-time backend
- **Multiplayer gaming engine** (Pong, Tank Wars) built directly into the chat interface
- Game state synchronized between all clients and the Python backend

## Highlights

The multiplayer games running inside a chat interface was the standout feature — no separate app, no redirects. You're mid-conversation and you just... start a game.
