# Frameworks & Tools

[[index|Back to Home]]

## Frameworks

### React.js / Next.js
Frontend. [[Projects/RepoBrowser|RepoBrowser]] is built on Next.js + TypeScript, and [[Projects/Cocoon|Cocoon]] uses React with Vite for a more app-like experience.

### Spring Boot
Java backend. Powers the [[Projects/WiFiAutologger|WiFiAutologger]] server.

### FastAPI / Express.js
Python and Node API layers. [[Projects/Cocoon|Cocoon]] uses FastAPI as the backend for auth and journaling flows.

### Flutter
Mobile UI. Used in [[Projects/WiFiAutologger|WiFiAutologger]] for the cross-platform mobile app.

### WebSockets
Real-time comms. Core to [[Projects/RetroBBS|RetroBBS]] for messaging and game state sync.
